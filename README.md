# 星图工具
游戏内按F1键打开或关闭GUI<br>
在游戏内,按F2键传送至目标恒星或星球<br>
下载链接 https://github.com/dsp-mod/StarMapTools/releases/download/v2.0/StarMapTools_2.0.zip<br>
演示视频 https://www.bilibili.com/video/BV1Vp4y1W7EN/
